# Files

### Open file inside Vim
`:e /path/to/file`

# Windows (Panes)
`<Ctrl>+w`

### Switch Window
`<Ctrl>+w <direction arrow>`

### Close Window
`<Ctrl>+w q`

# Tabs
`<Ctrl>+t`

### Switch Window
`<Ctrl>+t <direction arrow>`

# Editing

## Undo and Redo

### Undo
`u`

### Redo
`<Ctrl>+r`

## Copy and Paste

### Select multiple lines
Enables visual block to select multiple characters/lines
`<Ctrl>+v`

### Copy
`y`

### Cut
`dd`
